# Training Program

A personal, agent-assisted program to build technical fluency for software/AI product roles.

**Background:** Economics, entrepreneurship, strategy.  
**Goal:** Strong enough to ship small systems, talk credibly in technical rooms, and build AI products.

## How to use this

1. Open [docs/curriculum-map.md](docs/curriculum-map.md) — find the first `not started` module.
2. Open that module file in `docs/modules/`.
3. Work through it: read the concepts, do the exercises, tick the checks.
4. When all checks pass, mark it `complete` in the curriculum map and log the session.
5. Move to the next module.
