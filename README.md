# Training Program

A personal, agent-assisted program to build technical fluency for software/AI product roles.

**Background:** Economics, entrepreneurship, strategy.  
**Goal:** Strong enough to ship small systems, talk credibly in technical rooms, and build AI products.

## How to use this

1. Open [docs/CURRICULUM-MAP.md](docs/CURRICULUM-MAP.md) — find the current phase.
2. Open that phase's plan in `docs/phases/` — it tells you the recommended approach and sequencing.
3. Open the module file in `docs/modules/` — read the concepts, do the exercises, tick the checks.
4. When all checks pass, mark the module `complete` in the curriculum map and log the session.
5. Move to the next module. When all modules in a phase are done, move to the next phase plan.
